# ElderHome Hub - ESP32-S3-BOX-3 Prototype

A home-based smart assistant prototype for elderly users, built using ESP32-S3-BOX-3, ESP-IDF, and Wi-Fi.  
This prototype demonstrates **physical button detection, touchscreen UI, and HTTP POST notifications** to a server or web app.

---

## Features

- Detects physical button presses
- Sends HTTP POST requests when buttons are pressed
- Connects to Wi-Fi network
- Basic LCD touchscreen setup (starter code for UI)
- Ready to extend for voice, big-button UI, and notifications

---

## Hardware

- **ESP32-S3-BOX-3**  
- Onboard buttons or external jumper wire buttons  
- Built-in LCD touchscreen  
- Wi-Fi connectivity
- **Jumper-wires**
- **MPU-6050**

---

## Requirements

- VSCode with **ESP-IDF Extension** installed  
- ESP-IDF framework (latest version, installed via EXPRESS mode)  
- USB cable to connect ESP32-S3-BOX-3  
- Wi-Fi network credentials  
- Optional server or webhook to receive HTTP POST requests
